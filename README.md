# IOS_Native_NotesApp

This app building only for learning reasons, this app still building and will change, and i will add new screenshots, and I will try implement all funcionality represented in original app, I think it will be nice practice for me)

# Usage

Download Project folder and open folder path in terminal, type ```pod install```, it will install all necessary pods that i have in my Podfile, installed and used, and just run simulator and run the app from XCode
